# Runtime

Domain: Runtime

## Purpose

Define dev.kit as the execution surface that enforces CDE constraints.

## Responsibilities

- Discover repository-defined capabilities.
- Expose stable, versioned commands.
- Enforce schemas and constraints.
- Execute workflows safely.
- Capture state and outputs.

## Authority Model

- Reasoning systems: analyze, propose, explain.
- dev.kit: validate, execute, persist.

Execution authority is never delegated.
