# Codex Workflows

Workflow: codex.rules
- Purpose: manage Codex rules from dev.kit docs.
- Steps:
  1) `dev.kit codex rules --plan`
  2) `dev.kit codex rules --apply`
  3) `dev.kit codex rules --show`

Notes
- Always review the plan output before apply.
- Rules are generated from docs sources and templates.
